## CatCulator
저희의 앱을 이용하여 축제의 정산 과정의 간편화와 축제 상품을 쉽게 둘러볼 수 있습니다!

## 판매자 기능
- 판매자 화면
  - 현장 판매와 온라인 판매가 구분되어 있습니다.
<img src="https://github.com/user-attachments/assets/2995bd3a-4f17-4f20-8dbe-03c01b271240" alt="판매자 화면" width = "300" height = "600"/>

  - 현장 판매를 선택하신 경우, 현재 있는 부스를 고르시거나 하단의 부스 새로 추가하기 버튼을 누르셔서 부스를 추가하시면 됩니다.
<img src="https://github.com/user-attachments/assets/f31b7963-b80b-4df0-bfa4-c6e87508f091" alt="온라인 부스" width = "300" height = "600"/>

  - 부스를 추가하실 경우에 축제명, 부스명, 참여 작가 명단, 부스 위치를 기입하시면 됩니다.
  - 사전 판매를 실시할 경우에는 사전 판매 실시 여부를 체크하시면 됩니다.
  - 모든 정보를 입력하시면 우측 하단의 확인 버튼을 누르시면 부스가 추가됩니다.
<img src="https://github.com/user-attachments/assets/485a0a14-5797-4b1e-943e-67173418dccb" alt="부스 추가" width = "300" height = "600"/>
  
- 부스 관리
  - 현장의 부스에서 판매할 상품을 관리하고, 현장에서 물건을 팔 때에 손쉽게 계산을 할 수 있도록 돕습니다.
  - 상품을 누를시에는 구매자에게서 주문을 받는 현장 주문으로 넘어갑니다.
  - 상품편집을 누를시에는 판매할 상품을 추가하거나 수정하는 화면으로 넘어갑니다.
  - 상단의 정산하기를 누를시에는 축제가 끝나고 작가들에게 판매 금액을 정산하는 화면으로 넘어갑니다.
  - 상단의 사전구매를 누를시에는 사전구매를 사전구매를 한 구매자들에게 주문번호를 받는 화면으로 넘어갑니다.
  - 우측 상단의 판매기록을 누를시에는 현재까지 팔린 상품들의 기록을 보실 수 있습니다.
  - 상품 검색창을 이용하셔서 찾으시는 상품을 빠르게 검색하실 수 있습니다.
  - 작가 전체 필터를 이용하셔서 특정 작가의 상품만을 보실 수 있습니다.
<img src="https://github.com/user-attachments/assets/acb6951b-7424-426f-9cad-e68c58211701" alt="부스 상품 판매" width = "300" height = "600"/>

- 상품 관리
  - 상단의 상품 검색창을 이용하여 빠르게 찾으시는 상품을 검색하실 수 있습니다.
  - 그 아래의 작가 전체 필터를 이용하셔서 특정 작가의 상품만을 보실 수 있습니다.
  - 상품을 누르셔서 상품의 정보를 보거나 수정하실 수 있습니다.
  - 우측 하단의 + 버튼을 누르셔서 상품을 추가하실 수 있습니다.
<img src="https://github.com/user-attachments/assets/e4ddc719-32f0-4fdf-90e8-d60c112ca3e5" alt="상품 관리" width = "300" height = "600"/>
<img src="https://github.com/user-attachments/assets/3bb6213b-2ab3-4600-ad85-511c95ec1cee" alt="상품 정보" width = "300" height = "600"/>

  - 상품명, 판매 작가명, 원가, 판매가, 재고 수량, 상품 종류을 입력하시면 됩니다.
  - 사진 업로드로 상품의 사진을 올리실 수 있습니다.
  - 만약 사진을 올리시지 않으면 기본 이미지로 지정됩니다.
  - 모든 정보를 입력하신 후에 우측 하단의 확인 버튼을 누르시면 상품이 추가됩니다.
<img src="https://github.com/user-attachments/assets/5a8372d7-5c1f-487f-9977-751cacf28c0b" alt="상품 추가1" width = "300" height = "600"/>
<img src="https://github.com/user-attachments/assets/354943da-def9-45e1-a24b-690f8ae0be84" alt="상품 추가2" width = "300" height = "600"/>

- 현장 주문
  - 구매자가 원하는 주문 목록을 확인하는 화면입니다.
  - - + 버튼을 이용하여 수량을 수정하실 수 있습니다.
  - 우측 하단의 결제하기 버튼을 누르시면 현금 결제로 넘어갑니다.
<img src="https://github.com/user-attachments/assets/4b3da18d-432d-48c5-921e-b3cdf8d2ba7f" alt="pos 주문" width = "300" height = "600"/>

- 판매 기록
  - 판매 기록 화면에서는 여태까지 팔린 상품들을 보실 수 있습니다.
  - X 버튼을 누르시면 판매 기록을 삭제하실 수 있습니다.
<img src="https://github.com/user-attachments/assets/fec7d1bf-162b-45b2-a366-29b0f5a9010d" alt="판매 기록" width = "300" height = "600"/>

- 부스 정산
  - 부스 정산 화면에서는 각 작가에게 얼마나 정산을 해야하는지 표기가 됩니다.
  - 작가를 누르시면 그 작가의 상품들 판매 현황 자세히 확인하실 수 있습니다
<img src="https://github.com/user-attachments/assets/67da8947-0a26-4271-85e2-82f1e6884e55" alt="정산 기능" width = "300" height = "600"/>

- 사전 구매
  - 사전에 구매한 구매자들의 주문번호를 QR코드로 확인하여 주문을 진행할 수 있습니다.

- 온라인(통신) 판매
  - 품절된 상품들을 수요 조사를 거친 후에 온라인 판매로 전환하여 판매 할 수 있습니다.

  - 온라인 상품들을 온라인 판매 화면에서 확인하고 수정하시거나 추가하실 수 있습니다.
  - 우측 상단의 주문 목록을 통하여 온라인 상품 구매를 하신 구매자 분들의 연락처와 결제 목록을 확인하실 수 있습니다.

## 구매자 기능  
- 구매자 화면
  - 현장 구매와 온라인 구매가 구분되어 있습니다.  
<img src="https://github.com/user-attachments/assets/98cc0a7e-dd4f-451b-a035-0efb43256684" alt="구매자 화면" width = "300" height = "600"/>

  - 현장 구매를 선택하실 경우 빨간 원 안의 축제 선택 드롭다운을 누르시면 찾으시는 축제를 선택하신 후 아래의 확인 버튼을 누르시면 됩니다.
<img src="https://github.com/user-attachments/assets/e22a58eb-6e25-44a4-b107-5854005dcae2" alt="축제 선택" width = "300" height = "600"/>

<img src="https://github.com/user-attachments/assets/cc6b7521-6bd1-4fc1-bdcd-31349343867f" alt="온라인 축제 선택" width = "300" height = "600"/>

- 지도
  - 빨간색 원의 메인 부스 선택의 드롭다운을 누르시면 찾으시는 부스 열을 누르셔서 지도상에 표시하실 수 있습니다.
  - 빨간색 원의 편의시설 또는 존 선택의 드롭다운을 누르시면 찾으시는 장소를 찾아서 지도상에 표시하실 수 있습니다.
  - 지도를 확대나 축소하기 위해서는 두 손가락을 화면에 대고 모으거나 펼치시면 됩니다.
<img src="https://github.com/user-attachments/assets/9aa987fd-5745-47e7-86cf-4f6dfb5c71da" alt="지도" width = "300" height = "600"/>

<img src="https://github.com/user-attachments/assets/781086fb-85bd-4617-9ab3-0b2bacf6a75e" alt="부스 정보" width = "300" height = "600"/>

  - 사전 구매에서는 검색창으로 찾으시는 부스명이나 작가명을 검색하실 수 있습니다.

결제는 가상으로 진행이 되며 실제 결제는 이뤄지지 않습니다.  
안드로이드 에뮬레이터로 실행할 시에 권장되는 해상도는 1440x2560입니다.
