## CatCulator
저희의 앱을 이용하여 축제의 정산 과정의 간편화와 축제 상품을 쉽게 둘러볼 수 있습니다!

## 기능
판매자 기능
- 부스 POS 관리  
- 부스 정산
- 사전 구매
- 통신 판매
- 온라인 판매

구매자 기능
- 지도
- 부스, 상품 보기
- 사전 구매, 장바구니, 주문 목록
